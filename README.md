# ⏱️ ChronoX - Stopwatch Application

ChronoX is a responsive stopwatch web application that allows users to accurately measure time intervals using start, pause, reset, and lap functionalities.

## 🚀 Features

- Start Stopwatch
- Pause Stopwatch
- Reset Stopwatch
- Lap Time Recording
- Real-Time Millisecond Tracking
- Responsive Design
- Clean and Modern User Interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
